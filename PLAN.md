あなたは優秀なフルスタックエンジニアです。
Cloudflare Workers + Hono + TypeScriptを使って、「シンプルで美しい時差変換ツール」を作成したいです。

## 開発環境
- フレームワーク: Hono (v4系)
- ランタイム: Cloudflare Workers
- 言語: TypeScript
- スタイリング: Tailwind CSS (HTMLヘッダーでCDN読み込み)
- ライブラリ: date-fns, date-fns-tz (インストール済みと仮定)
- 描画: HonoのJSX (サーバーサイドレンダリング)

## 要件仕様
1. **画面構成**
   - 中央にカード型のUIを配置。
   - 上段：「基準となる日時」入力エリア（日付、時刻、タイムゾーン選択）
   - 中段：変換方向を示す矢印アイコンと「入替ボタン」
   - 下段：「変換後の日時」表示エリア（日付、時刻、タイムゾーン選択）
   - 最下部：「結果をコピー」ボタン

2. **機能**
   - ページを開いたとき、デフォルトで「日本時間 (Asia/Tokyo)」と「現在時刻」を基準に入力済みにする。
   - 変換先はデフォルトで「協定世界時 (UTC)」または「ニューヨーク (America/New_York)」にする。
   - 日付や時刻を変更したら、JavaScriptで即座に計算して下段の結果を書き換える（ページリロードなし）。
   - 「結果をコピー」ボタンを押すと、クリップボードに `2023-10-10 10:00 JST = 2023-10-09 21:00 EST` のような形式でコピーし、コピー完了トーストを表示する。

3. **タイムゾーンの選択肢**
   - 全てを羅列せず、主要なものに絞る（JST, UTC, EST, PST, CST, CET, GMT, IST, CST(中国), AUDなど）。
   - わかりやすく「🇯🇵 東京 (Asia/Tokyo)」のように国旗の絵文字をつける。

4. **デザイン (Tailwind CSS)**
   - 背景は目に優しい薄いグレーかオフホワイト。
   - メインカードは白背景、角丸、ドロップシャドウ付き。
   - 文字サイズや余白を十分に取り、スマホでも操作しやすくする（レスポンシブ対応）。
   - 昼夜がわかるように、時刻の横に☀️や🌙のアイコンを表示するロジックを入れる（6-17時を昼とするなど簡易的でOK）。

## 出力依頼
- `src/index.tsx` のコードをすべて書いてください。
- クライアント側の処理（計算やイベントリスナー）は、`script`タグの中に `raw` 文字列として埋め込む形式で実装してください（HonoのJSXでクライアントJSを動かす一般的な手法）。
